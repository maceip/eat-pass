Platform surfaces

The operator interface is the VerificationPolicy JSON file. The attester --gate selects how evidence arrives; allow selects who may mint. Same mint protocol on every surface — attest → FAEST → PoMFRIT → spend.

No “gauges”. Runtime controls are gates (crypto verifier backend: azure, desktop-tpm, …) and policy (appraisal checks after verify). Privileged tools add tool-gate as a spend-time RP (email, etc.).
